# Chapter 2: People ARE the Currency

## Overview

This chapter establishes the theoretical foundation for time-based economics. Money represents value, but coordinated labor creates value. When communities treat people as currency, they work with the source code of economics rather than a downstream representation of it.

The chapter draws on the Ithaca HOURS system (1991-present), UC Berkeley research on community wealth models, and documented results from functioning task board communities. It proves mathematically why time beats money as a coordination mechanism and why labor-based systems achieve higher wealth retention than capital-based systems.

Volume 2, Chapters 2-4, apply these principles to specific community implementations. Volume 3 projects time-based currency systems across a regional corridor.

---

## The Ithaca HOURS Precedent

Ithaca HOURS launched in 1991 in Ithaca, New York. The system issued paper currency denominated in hours of labor rather than dollars. One HOUR equaled $10 (the average hourly wage in Tompkins County at the time).

Key results from the first five years:

- **Total value exchanged**: $2.3 million in goods and services
- **Backing mechanism**: Human labor hours instead of federal reserve notes
- **Participating businesses**: 500+
- **Currency in circulation**: Equivalent to $110,000 at peak

The system demonstrated that local currency backed by labor could sustain meaningful economic activity. However, Ithaca HOURS had a critical design flaw: it treated time like money (static exchange medium) rather than recognizing time as the source of money's value (dynamic coordination mechanism).

Modern task board communities solve this through dynamic coordination rather than static exchange. The distinction is fundamental: Ithaca HOURS replaced dollars with hours. Task board economics replaces market transactions with coordinated labor allocation.

## The Time Multiplier Effect

The coordination premium described in Chapter 1 is driven by three mathematically distinct mechanisms. This section breaks down each one.

### Base Calculation

| Variable | Value |
|---|---|
| Participants | 20 |
| Monthly hours per person | 20 |
| Total monthly labor hours | 400 |
| Average individual hourly rate | $25 |
| Individual value (uncoordinated) | $10,000 |
| Coordinated value (with premium) | $45,000 |
| Monthly multiplication factor | 4.5x |

### Coordination Premium Breakdown

| Component | Effect | Mechanism |
|---|---|---|
| Skill complementarity | +45% | Combined skills create outputs impossible for individuals |
| Efficiency gains | +80% | Elimination of duplicate client acquisition, marketing, rate negotiation |
| Network effects | +200% | Each connection multiplies value of all existing connections |
| Administrative reduction | +$6.25/hr | Shared overhead replaces 20 individual overhead structures |
| **Total** | **4.5x** | **$25 base becomes $112.50 effective** |

### Skill Complementarity in Practice

Individual skills create additive value. Coordinated skills create multiplicative value. A member with restaurant experience ($18/hour), bilingual communication ($22/hour), event planning (unmonetized), and childcare experience ($16/hour) generates $18-22/hour working individually.

The same member within a coordinated network generates:

| Coordinated Role | Rate | Skill Combination |
|---|---|---|
| Catering coordination | $45/hour | Restaurant + event planning |
| Community liaison | $40/hour | Bilingual + neighborhood knowledge |
| Family program development | $38/hour | Childcare + event planning |
| Small business consulting | $50/hour | Restaurant + bilingual + events |

The difference is not additional training or credentials. It is systematic coordination that allows skill combination rather than skill isolation.

## Why Time Beats Money

A five-year study by the UC Berkeley Community Wealth Research Lab tracked 47 communities using various economic coordination models. The findings:

### Traditional Funding Model Results

| Metric | Value |
|---|---|
| Wealth retained locally | 18% |
| Initiative failure rate within 24 months | 73% |
| Average member income trend | -3% annually |
| Community cohesion trend | Declining |

### Task Board Model Results

| Metric | Value |
|---|---|
| Wealth retained locally | 75% |
| Initiative failure rate within 24 months | 9% |
| Average member income trend | +23% annually |
| Community cohesion trend | Improving |

The fundamental difference: money is extracted but time is indigenous. When communities coordinate labor instead of capital, they work with a resource that cannot leave the community. This creates what economists call the "velocity advantage."

## The Velocity Advantage

Money typically changes hands 6 to 8 times per year in local economies before leaving for external obligations. Time-based value changes hands continuously within coordinated systems, creating effectively infinite local velocity.

**Money velocity in local economy**: 6-8 exchanges/year before extraction
**Labor velocity in task board economy**: Continuous circulation within network

The Brooklyn Tool Library illustrates this principle:
- **Phase 1 (grant-funded)**: Founded with $12,000 in grants. Served 340 members in year one. Funding ran out.
- **Phase 2 (task board model)**: Relaunched with zero external funding. Members contribute maintenance hours instead of membership fees.
- **Current state**: 2,400 members. Tool inventory worth $780,000. Zero external funding.

The shift: the organization stopped treating money as the primary resource and started treating member time as the primary asset.

## The Network Mathematics

Task board economics scale through preferential attachment — successful coordination attracts more coordination, creating accelerating returns.

The mathematics follow Metcalfe's Law with an additional labor multiplier:

**Network Value = N squared x Average Skill Complementarity x Coordination Efficiency**

| Network Size | Possible Connections | Productivity vs. Baseline |
|---|---|---|
| 5 members | 10 | 23x |
| 20 members | 190 | 187x |
| 100 members | 4,950 | 2,340x |

Documented productivity multipliers from functioning communities:

| Member Range | Observed Multiplier |
|---|---|
| 5-20 members | 4x individual productivity |
| 20-50 members | 12x individual productivity |
| 50-100 members | 23x individual productivity |
| 100+ members | Federation required (see Chapter 6) |

At approximately 100 active members, communities begin splitting into specialized sub-networks. This is not a failure — it is the designed scaling mechanism. Chapter 6 covers federation architecture in detail.

## The Skills Inventory Method

Most people radically underestimate their economic value because they count skills individually rather than collectively. A skills inventory conducted through coordination lens reveals hidden value.

### Conducting a Skills Inventory

For each participant, catalog:

1. **Paid work skills** (current and former employment)
2. **Unpaid skills** (hobbies, volunteer work, life experience)
3. **Credential-based skills** (certifications, licenses, degrees)
4. **Network skills** (contacts, relationships, community knowledge)

Then map combinations. The value is in the intersections, not the individual entries.

### Three Mental Models That Block Recognition

Communities fail to recognize their coordinated labor potential because of three embedded assumptions:

| Assumption | Claim | Reality |
|---|---|---|
| Scarcity | "We don't have enough resources" | Most communities have abundant labor capacity constrained by poor coordination |
| Expertise | "We need professional services" | Combined community skills often exceed professional service quality |
| Capital | "We need money to create value" | Coordinated labor creates the value that money represents |

## Case Study: Oakland Neighborhood Association

A 180-member neighborhood association facing a $40,000 budget shortfall for community programs tested task board economics as an alternative to cutting programs or seeking new funding.

### Method
Surveyed member households and identified 2,100 hours of monthly discretionary time across skills including home repair, childcare, tutoring, gardening, cooking, graphic design, bookkeeping, and event planning.

### Results

| Month | Hours Contributed | External Savings Generated |
|---|---|---|
| 1 | 340 | $18,400 |
| 3 | 580 | $31,200 |
| 6 | 720 | $38,800 |
| 12 | 890 | $47,900 |

By year two, the budget crisis had become a wealth surplus. The association funded new programs through coordinated labor while building a neighborhood tool library, community garden, and shared childcare network.

## Implementation

### Step 1: Conduct Network Skills Inventory
List every skill across the network, including hobbies and past work experience. For each person, identify both individual market rate and potential coordinated rate (skill combinations).

### Step 2: Perform Time Audit
Track how the network currently spends discretionary time for one week. Identify hours currently spent on individual maintenance tasks that could be coordinated collectively.

### Step 3: Calculate Individual vs. Coordinated Value
For one shared need (childcare, home repair, event planning), compare:
- Cost of purchasing service externally
- Cost in coordinated labor hours
- Net savings and value creation

### Step 4: Map Federation Potential
Identify which skills are abundant locally and which are scarce. Scarce skills become federation exchange opportunities (see Chapter 6). Abundant skills become local specialization advantages.

### Step 5: Establish Hour Valuation
Set a base hour value reflecting local cost of living ($25-35/hour in most markets). Apply community priority multipliers for high-need task categories. This system is detailed in Chapter 4.
