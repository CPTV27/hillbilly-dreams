# How Do I Update the Website?

## Steps

1. **Turn on Edit Mode** — Visit `/api/draft?enable=1` on any page of the website. This puts the site into editing mode.
2. **Click on text to edit it** — Any text on the page becomes editable. Just click on it and start typing.
3. **Click on images to swap them** — Click any image to replace it with a new one from the Media Vault.
4. **Click Save** — When you're happy with your changes, click the Save button. Your edits go live.
5. **Turn off Edit Mode when you're done** — Visit `/api/draft?enable=0` to go back to normal viewing.
