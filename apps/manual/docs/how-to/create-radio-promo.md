# How Do I Create a Radio Promo?

## Steps

1. **Go to the Content Studio** — Open `/admin/studio` in your browser.
2. **Pick "Radio Spot"** — Select Radio Spot from the content type list.
3. **Type in the show or event details** — Include the artist name, date, time, venue, and anything else listeners need to know.
4. **Review the script** — Read through the generated radio spot script. Make sure it sounds natural and has all the right info.
5. **Export the script** — Download or copy the script so you can record it or hand it off to the DJ.
