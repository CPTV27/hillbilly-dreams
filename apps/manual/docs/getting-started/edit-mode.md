# Edit Mode

Edit Mode lets you change text and images on any page of the website — right in the browser.

## How to Turn It On

Visit this URL in your browser:

```
bigmuddytouring.com/api/draft?enable=1
```

You'll see a gold toolbar appear at the top of the page. You're now in Edit Mode.

## How to Edit

### Text
- Click any text on the page
- A dashed gold border appears around it
- Type your changes
- The text updates in real time

### Images
- Click any image
- You'll be prompted for a new image URL
- Paste a URL from the Media Vault
- The image swaps immediately

## How to Turn It Off

Visit:

```
bigmuddytouring.com/api/draft?disable=1
```

Or click "Exit" on the toolbar.

## Important Notes

- Right now, edits are **previews only** — they show you what the change will look like but don't save permanently yet. Chase is wiring up the save feature.
- When the save feature is live, **double-check before saving**. This changes the real website.
- If you mess something up, just refresh the page to go back to the original.
