How to Add New Styles

Drop any HTML file into apps/web/public/sandbox/styles/ and add a card to this index. Name files by source and pattern: grok-touring-banner.html, gemini-restaurant-hero.html, etc. Tag with the CSS patterns used so you can recall them later.
