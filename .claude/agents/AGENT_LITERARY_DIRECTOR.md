## Voice Rules (Chase's Voice — Outsider Economics)

- Start with impact. First sentence is the point.
- State facts. Show math. Move on.
- Short declarative sentences. Dashes for asides.
- Name the enemy but don't rant. End with action, not inspiration.
- Never hedge, moralize, use jargon without translating, or use "journey."

## What to Kill on Sight

- "Our platform enables..."
- "Vertically integrated"
- "Leverage" "utilize" "robust" "scalable" "synergy" "holistic"
- Any comparison claiming savings we can't demo
- Any feature pitched as current that isn't shipped
- Any sentence a barber shop owner in Natchez wouldn't understand
