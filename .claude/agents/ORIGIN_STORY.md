## The Origin

Chase designed a complete media production-to-distribution pipeline in 2022 called the DeFacto Codec Market. It was a vision for how to create an entire media economy — broadcast, production, analytics, distribution, audience measurement — using open source tools (OpenBroadcaster) and cloud infrastructure.

The insight: the same organizational structure that runs a Viacom or a BBC can run a small-town media ecosystem. You don't need to be big. You need to be organized.

## The Proof

- **daytime.tv.us** — Prototype of a local TV channel for public access in Woodstock, NY. Streaming + YouTube + public access. The first implementation of the concept.
- **mansplaination.tv** — The original content brand.
- **Big Muddy** — The Mississippi corridor implementation. Same architecture, different market, fully operational.

## The Architecture (Same at Both Scales)

| Layer | Global Scale | Main Street Scale |
|---|---|---|
| Broadcast Platform | OpenBroadcaster (P2P network) | OpenBroadcaster (local radio, 18 shows) |
| Cloud Processing | Codec Market (transcoding) | Vertex AI (generation pipeline) |
| Analytics | deFacto Global (AI backend) | Multi-model AI routing + Delta Dawn |
| Asset Management | ART-Chive (blockchain DAM) | GCS + Media Vault + hot folder sync |
| Production | Media Production & Programming | Content Studio (one input → 5 channels) |
| Catalog | Content database | Directory listings + articles + knowledge base |
| Distribution | Smart TVs, broadcast, film, social | Plex in-room TV, web radio, social, magazine |
| Audience Data | Nielsen, Gracenote | First-party analytics, review monitoring |
| Marketplace | NFT making machine | Photography storefront + Stripe Connect |
| Public Access | daytime.tv.us | bigmuddyradio.com + YouTube + streaming |

## The Value Propositions

1. **Any small town can have a media ecosystem.** Not just a Facebook page — a magazine, a radio station, a directory, a streaming channel, an entertainment pipeline. The tools exist. The architecture is proven.

2. **AI replaces the headcount, not the roles.** A media company needs 10+ roles. AI handles 70-90% of each role. 3 humans + AI = a full media operation.

3. **The infrastructure is the product.** MBT doesn't sell software — it sells the entire media stack as a service. $99/mo gets you what would cost $50K/mo to build yourself.

4. **Public access is the on-ramp.** Get on the air locally (public access TV, FM transmitter, web stream), then scale to streaming, social, YouTube. Same content, multiple distribution endpoints.

5. **The corridor is the content.** Memphis to New Orleans isn't just a route — it's a content engine. Every town, every musician, every restaurant is a story. The stories feed the magazine, the radio, the directory, the economy.
