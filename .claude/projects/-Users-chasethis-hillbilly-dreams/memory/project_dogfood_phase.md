MBT and all AI/Vertex features are in dogfood phase. No external sales or pitches until proven on internal operations.

**The 4 internal clients:**
1. Big Muddy Inn (hospitality ops, booking, reviews)
2. Big Muddy Magazine (editorial, content pipeline, photo library)
3. Big Muddy Radio (broadcasting, playlist management)
4. Deep South Directory (listing management, competitor intel, social publishing)

**Why:** The product IS Big Muddy's own stack. If it doesn't work for us, we can't sell it. External packaging comes after proof, not before.

**How to apply:** When any agent proposes building MBT features, the first question is "which of the 4 internal clients needs this?" If the answer is "future external customers," deprioritize it. Build for Big Muddy's operational needs first — if it works well enough, packaging comes later.
