# `packages/modules/booking/`

## MBT Booking Module (Phase C Block 2)

This README details the core Booking module for the Montana Business Toolkit (MBT) monorepo.

---

### 1. Purpose

The `booking` module serves as the central engine for managing various types of bookable resources, inventory, and associated workflows across the Montana Business Toolkit's diverse business units. It owns the logic for ticket inventory management and sales (e.g., Big Muddy Inn Blues Room shows), the comprehensive private event quote-to-deposit process (e.g., Inn weddings, retreats, and dinners), service appointment scheduling (e.g., Tuthill listing shoots, photography sessions), and the complex artist hold/confirm negotiation workflow for Big Muddy Touring. While it integrates with and wraps existing functionality for Inn room bookings via Cloudbeds (treating Cloudbeds as the system of record for those specific resources), this module is the definitive source of truth and orchestrator for all other inventory, quotes, and direct sales within the MBT ecosystem. It provides a unified API for managing availability, reservations, and conversions across different booking types, ensuring data consistency and streamlined operations.
