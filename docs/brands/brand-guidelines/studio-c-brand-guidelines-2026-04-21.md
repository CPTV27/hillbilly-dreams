# Studio C — Brand Identity Guidelines

**Spec version:** 2026-04-21
**Brand owner:** Tuthill Design LLC (Studio C — production DBA; Elijah Davis, operator)
**Voice doc:** `docs/voice/studio-c.md`
**Intended designer:** Miles Dubois + Elijah Davis (in-house — sister DBA to Tuthill Design, visually related but distinct)
**Template reference:** Rosemary Hallmark Creative / The Big Muddy Inn + Blues Room, March 2023

> **Positioning premise from the voice doc:** *"Direct & action-oriented, prepared & proactive, technically proficient, confident & capable, reliable & collaborative."* Studio C is the **production arm** — broadcast, events, concert-capture, multi-cam, live-stream, post-production. The identity is technical, rigorous, unmistakably professional, and visually reads as "control room."

## 3. Primary logo — Signet + Wordmark

Two elements, locked:

- **Signet:** a bold outlined `C` inside a precise circle, line weight equivalent to the wordmark's stem width. Reads like an aperture ring, a lens element, or a "cue" light. Rendered in MBT Gold (primary) or Console Black (inverse).
- **Wordmark:** `STUDIO C` in Input Mono Bold (or Roboto Mono Bold as substitute), all caps, tight letter-spacing, set to the right of the signet.
- The lockup has a fixed relationship: the signet's diameter equals the cap-height of `STUDIO C`, with padding equivalent to half a cap-height between them.

## 4. Secondary logo — Signet alone

- The `C` signet by itself, used wherever the wordmark won't fit or isn't needed: favicons, lens-cap stickers, flight-case labels, Slack/Discord avatars, video bug lower-right during livestreams.
- Color treatments: MBT Gold on Console Black (primary), Console Black on MBT Gold (inverse), Cloud Dancer on Console Black (neutral white version for broadcast graphics).

## 5. Brand colors

| Role | Name | Pantone | Hex | CMYK | RGB |
|------|------|---------|-----|------|-----|
| Primary dark | **Console Black** | Black 6 C | `#0A0D12` | 85 / 75 / 60 / 85 | 10 / 13 / 18 |
| Primary accent | **MBT Gold** | 7753 C | `#B8860B` | 25 / 45 / 100 / 15 | 184 / 134 / 11 |
| Signal warning | **Signal Amber** | 1385 C | `#C8852A` | 20 / 50 / 100 / 5 | 200 / 133 / 42 |
| UI utility | **Signal Gray** | Cool Gray 8 C | `#8E8E8E` | 25 / 20 / 20 / 50 | 142 / 142 / 142 |
| Text on dark | **Cloud Dancer** | 11-4201 TPG | `#F1F0EC` | 4 / 3 / 6 / 0 | 241 / 240 / 236 |

Rationale: Studio C is a **control room at 4 AM during a live show.** Console Black dominates because that's what production gear looks like. **MBT Gold** is the shared accent with the wider MBT platform (Studio C is platform-tier production; the gold ties Studio C visually to the MBT modules it serves). **Signal Amber** is reserved for the same thing it means on a real broadcast desk — "caution, attention required" — used sparingly on alert states, error messages, red-light on-air indicators.

Proportions: 75% Console Black, 15% MBT Gold (logo + primary buttons + headlines), 5% Cloud Dancer (body type on dark), 3% Signal Gray (UI secondary), 2% Signal Amber (alerts + "ON AIR").

## 6. Incorrect usage

- Do **not** use consumer-tech gradients (iridescent, neon, holographic) — Studio C is industrial broadcast, not "creator economy."
- Do **not** place the signet without its precise circle — the geometry is the signature.
- Do **not** use decorative or humanist fonts — monospace + geometric sans only.
- Do **not** swap MBT Gold for a yellow or orange that isn't the exact Pantone 7753 C — family coherence matters.
- Do **not** display the Signal Amber "ON AIR" indicator outside of actual live contexts (no fake urgency).
- Do **not** use the `C` signet on its own at a size where the circle doesn't render cleanly (minimum 24px screen, 6mm print).
- Do **not** combine Studio C identity with Tuthill Design identity on a single piece unless the piece is a **Tuthill × Studio C** explicit co-brand (see Tuthill guideline §11).

## 7. Typography

| Role | Typeface | Weights | Usage |
|------|----------|---------|-------|
| Display (wordmark) | **Input Mono** (or Roboto Mono) | Bold | Wordmark only |
| Headline | **Söhne Breit** (or GT America Extended) | Semibold | Section heads, deliverable titles |
| Subhead | **Söhne** | Regular | Deck copy, subsection headers |
| Body | **Söhne** | Regular / Buch | Web body, proposal body, production binder body |
| Data / timecode / specs | **Input Mono** | Regular / Medium | Production specs, file names, timecodes, signal chains |

Rule: anywhere a number represents **time, resolution, or signal** (timecode, 1080p60, 48kHz, 200A, 600ms), set it in Input Mono.

## 8. Submark + production marks

1. **C signet** (see §4).
2. **ON-AIR mark:** a Signal Amber dot + `ON AIR` in Input Mono small caps, used during live broadcasts and in lower-third graphics.
3. **Signal-chain mark:** a horizontal Input Mono line `[SRC] → [MIX] → [ENC] → [CDN]` depicting the broadcast signal flow; used as a decorative technical element on deliverable covers and production binders.
4. **Deliverable slate:** a 16:9 box with `STUDIO C · [CLIENT] · [PROJECT ID] · [TAKE ##]` in Input Mono — used as the first frame of every deliverable video and the first page of every production document.
5. **Tier mark:** `TIER 01 / TIER 02 / TIER 03` set in Input Mono for service tiers (see voice doc §Conventions).

## 9. Patterns

- **Pattern A — Waveform Envelope:** a horizontal amplitude envelope rendered as a fine Signal Gray stroke on Console Black, used as a footer decoration on title pages and email headers.
- **Pattern B — Patch Bay:** a grid of evenly-spaced small circles (representing patch-bay jacks) in Signal Gray on Console Black, 8mm grid, behind production-binder cover plates.
- **Pattern C — Timecode Strip:** a horizontal strip with timecode tick marks (`01:02:03:04`) in Input Mono Signal Gray, used as a section divider in proposals and post-production delivery docs.
