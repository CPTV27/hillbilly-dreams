## 3. Economics — three clean options

### Option A — **Contractor 1099 + rev share on upsell**
- Base day rate or project fee for Tuthill / HDI work.
- **Rev share** on net new MRR (e.g. Deep South Directory upgrades, recurring media retainers) that the Northeast team originates.
- **Pros:** Fast to stand up. **Cons:** No long-term alignment if rates are capped.

### Option B — **Regional LLC + profit split**
- Northeast delivery runs through a single LLC (operators as members).
- HDI / Tuthill contracts **with the LLC** for white-label production.
- **Pros:** Clean invoicing, scalable hires. **Cons:** Legal + accounting setup cost.

### Option C — **Micro-equity / phantom equity in a subsidiary**
- Small equity or phantom pool tied to **Bearsville Creative** revenue milestones.
- **Pros:** Same moral contract as Tracy/Amy story. **Cons:** Cap table work; needs counsel.
