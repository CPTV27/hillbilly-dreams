## The Vision

Two locations. Both self-sufficient. Both can produce anything from a TikTok to a feature documentary. Cameras, lights, sound, switching, streaming, editing, color grading, audio mixing, music recording, distribution — all in-house. No rentals. No outsourcing. No excuses.

**Natchez:** The Big Muddy Inn — Victorian mansion. Hotel + bar + music venue + studio.
**Bearsville:** Utopia Studios — historic recording studio. API AXS console + tie-lines to Bearsville Theater.
