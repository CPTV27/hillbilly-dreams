## How to Create Each Shortcut

1. Open **Shortcuts** app on iPad
2. Tap **+** (top right)
3. Name it (e.g., "Show me the constellation")
4. Tap **Add Action**
5. Search for **Open URL**
6. Paste the URL
7. Tap the shortcut name at top → **Add to Siri**
8. Record the phrase
9. Done
