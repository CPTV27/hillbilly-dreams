## 3 · What we sell

### Offering 1 — Physical records
- **What it is:** Vinyl + CD pressings of releases
- **What's included:** Record + sleeve/jacket art (often Chase photography) + insert
- **Price:** $___ vinyl · $___ CD
- **Packaging:** Single release; potential subscription / record-of-the-quarter
- **Sold via:** Records site · in-Inn gift shop · merch table at shows · partner record stores

### Offering 2 — Downloads
- **What it is:** Direct-purchase digital downloads
- **What's included:** Lossless + MP3 versions
- **Price:** $___ per album · $___ per single
- **Packaging:** Per release
- **Sold via:** Records site · Bandcamp likely

### Offering 3 — Streaming catalog
- **What it is:** Distribution to Spotify · Apple Music · Tidal · YouTube Music etc.
- **What's included:** Playback royalties (split with artists per contract)
- **Price:** Free to listener; revenue per stream
- **Packaging:** Whole catalog distributed via DistroKid / TuneCore / CD Baby (TBD)
- **Sold via:** Streaming platforms

### Offering 4 — Merch tied to releases
- **What it is:** Shirts, posters, stickers tied to specific releases
- **What's included:** Per-product
- **Price:** $___ per item
- **Packaging:** Per release; bundle with vinyl pre-orders
- **Sold via:** Records site · merch table at shows

### Offering 5 — Sync licensing (future)
- **What it is:** Catalog tracks licensed for film, TV, ad, podcast use
- **What's included:** License rights · usage scope · term
- **Price:** Per-license (varies wildly by use)
- **Packaging:** Per request
- **Sold via:** Direct outreach + sync agents (when catalog is large enough)
