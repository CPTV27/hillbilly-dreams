# DEEP SOUTH DIRECTORY — Social Network Layer
## Feature Spec
## March 26, 2026

---

## WHAT IT IS

A peer-to-peer business network built into the Deep South Directory. Businesses refer customers to each other, build relationships, collaborate on promotions, and see who's sending them traffic — all inside the directory they're already on.

This is not LinkedIn for businesses. It's not a social media feed. It's **the trust network that already exists on Main Street, made visible and trackable.**

---

## WHY IT MATTERS

1. **Referrals already happen.** A guest asks the hotel where to eat. The hotel says "go to The Gumbo Pot." That referral is invisible right now — no one tracks it, no one benefits from it systematically. This makes it visible.

2. **The moat.** Once 20 businesses are referring customers to each other through the directory, none of them can leave without losing their referral network. That's lock-in through genuine value, not data hostage.

3. **Network intelligence.** Every referral is a signal. The platform learns which businesses complement each other, which corridors tourists follow, which combinations drive the most revenue. That intelligence feeds the AI for everyone.

4. **Local velocity.** A dollar that stays in town because a barber referred someone to the restaurant instead of them Googling a chain — that's the Outsider Economics thesis in action. The directory doesn't just list businesses. It keeps money circulating.

## THE FEATURES

### 1. Referrals

**What it does:** One business recommends another to a customer, tracked through the directory.

**How it works:**
- Business owner taps "Refer" on another business's directory listing
- Options: share a link via text/email to the customer, or just log the referral ("I told them to go there")
- The referred business gets a notification: "The Big Muddy Inn sent a guest your way"
- Both businesses see the referral in their network activity

**What the data shows over time:**
- "You've sent 34 referrals to The Gumbo Pot this quarter. They've sent you 12."
- "Your top referral sources: Big Muddy Inn (28), Natchez Visitor Center (15), Main Street Barber (9)"
- "Businesses you refer to but who haven't referred back to you yet" (gentle nudge to reciprocate)

**Tier placement:**
- Free: Can receive referrals. Can see who referred them.
- $20: Can make referrals. Can see basic referral activity.
- $99: Full referral analytics. Network graph. Reciprocity tracking. Referral leaderboard.

### 2. Introductions

**What it does:** A business owner connects two other businesses who should know each other.

**How it works:**
- "The new florist should meet the wedding coordinator at the Inn"
- Tap "Introduce" → select two businesses → optional note ("You two should talk about wedding packages")
- Both businesses get a notification with the introducer's name and context
- The introduction is logged in both businesses' network activity

**Why it matters:** This is how small town business networks actually grow — through introductions from trusted neighbors. The directory formalizes the behavior that already happens at chamber mixers and on front porches.

**Tier placement:**
- Free: Can receive introductions
- $20: Can make introductions
- $99: Can see introduction network (who's connecting who in your town)

### 3. Endorsements

**What it does:** A business publicly vouches for another business in the directory.

**How it works:**
- Not a review (customers write reviews). An endorsement is business-to-business.
- "I send all my guests to The Gumbo Pot. Best shrimp and grits in Natchez." — The Big Muddy Inn
- Endorsements appear on the endorsed business's directory listing
- Limited to businesses in the directory (no fake endorsements from outside)

**Why it matters:** An endorsement from a respected local business is worth more than 50 anonymous Google reviews. It's social proof from a peer, not a stranger.

**Tier placement:**
- Free: Can receive endorsements. Displayed on listing.
- $20: Can write endorsements for other businesses.

### 4. Collaboration / Group Promotions

**What it does:** Multiple businesses create a shared experience or promotion through the directory.

**Examples:**
- "Natchez Dinner Crawl" — three restaurants, one ticket, one evening
- "Stay & Play" — Inn booking + venue tickets + restaurant reservation bundled
- "Main Street Saturday" — 10 businesses run a coordinated sidewalk sale

**How it works:**
- Any $99 business can create a "collaboration" and invite other directory businesses
- The directory generates a shared landing page for the collaboration
- Revenue tracking per participating business
- Shared social media content generated automatically
- The AI suggests collaborations based on referral patterns: "You and The Gumbo Pot send each other a lot of traffic. Want to create a joint offer?"

**Tier placement:**
- $99 only: Create and manage collaborations
- $20: Can join collaborations created by $99 users
- Free: Can see collaborations but not participate

### 5. The Network Map

**What it does:** A visual map showing how businesses in a town are connected through referrals, introductions, and collaborations.

**How it works:**
- Each business is a node. Lines between them represent referral relationships.
- Thicker lines = more referrals. Color = direction (sending vs. receiving).
- Tap any connection to see the relationship history.
- The town's overall "network density" is a single number — how connected is Main Street?

**Who sees it:**
- Individual businesses see their own network (who they're connected to)
- CVB/Chamber partners see the full town network (anonymized referral volumes)
- Chase/admin sees everything (for case studies and product development)

**Tier placement:**
- Free: See your own connections (who referred you)
- $20: See your network neighborhood (your connections and their connections)
- $99: Full network map for your town. Network density score.
- CVB/Chamber: Town-wide view with aggregate data

### 6. Activity Feed

**What it does:** A simple feed showing network activity relevant to your business.

**What appears:**
- "The Big Muddy Inn referred a guest to you" (referral received)
- "Main Street Barber endorsed your business" (endorsement received)
- "You were introduced to Natchez Florist by Tracy at the Inn" (introduction)
- "Natchez Dinner Crawl launched — you're invited to join" (collaboration invite)
- "3 new businesses joined the directory in Natchez this week" (community growth)

**What does NOT appear:**
- Algorithmic content
- Ads
- Posts from businesses you're not connected to
- Anything that feels like a social media feed

**The feed is a notification log, not a timeline.** It shows what happened, not what someone posted. No likes. No comments. No engagement metrics. Just: here's what's happening in your business network.

**Tier placement:**
- Free: Basic activity (referrals received, endorsements)
- $20: Full activity feed
- $99: Activity feed + AI insights ("Your referral network grew 20% this month")

---

## WHAT THIS IS NOT

- **Not social media.** No posts. No likes. No comments section. No algorithmic feed. No engagement metrics.
- **Not LinkedIn.** No connection requests. No messaging. No "people you may know." No recruiter spam.
- **Not Yelp.** Customer reviews come from Google. This is business-to-business relationships, not consumer reviews.
- **Not a marketplace.** We don't take a cut of transactions between businesses. Referrals are free.
- **Not public.** The network data (who refers who, how much) is private to the businesses involved. The CVB/Chamber sees aggregates only.

## DATA MODEL (for build agents)

```
Referral {
  id
  fromBusinessId    — who made the referral
  toBusinessId      — who received it
  type              — 'sent_customer' | 'recommended' | 'logged'
  customerContext   — optional note ("guest looking for dinner")
  timestamp
  tracked           — did the customer actually visit? (future: receipt matching)
}

Introduction {
  id
  introducerId      — who made the introduction
  businessA         — first business
  businessB         — second business
  context           — "You two should talk about wedding packages"
  timestamp
  accepted          — did both businesses acknowledge?
}

Endorsement {
  id
  fromBusinessId    — who's endorsing
  toBusinessId      — who's being endorsed
  text              — the endorsement copy (max 280 chars)
  timestamp
  displayed         — is it showing on the listing?
}

Collaboration {
  id
  creatorBusinessId — who created it
  name              — "Natchez Dinner Crawl"
  description
  participants      — array of business IDs
  status            — 'draft' | 'active' | 'completed'
  landingPageUrl    — auto-generated
  startDate
  endDate
}

NetworkEdge {
  id
  businessA
  businessB
  referralCount     — total referrals in both directions
  lastActivity
  strength          — calculated from frequency and recency
}
```

## IMPLEMENTATION PRIORITY

1. **Referrals** — the core behavior. Ship this first.
2. **Activity Feed** — makes referrals visible and satisfying.
3. **Endorsements** — simple, high-value, easy to build.
4. **Network Map** — the visual that makes CVBs and chambers say "I need this."
5. **Introductions** — nice to have, not urgent.
6. **Collaborations** — the $99 tier killer feature. Build when there are enough businesses to collaborate.
