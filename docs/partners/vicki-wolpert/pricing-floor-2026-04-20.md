## The Directory engagement itself

Vicki's B2B Directory engagement has TWO revenue lines:

1. **Upfront fee** (~$15k per the 90-day plan baseline) — one-time project fee covering Directory module customization, content migration, launch coordination, initial listing setup
2. **Ongoing $500/mo retainer** — ongoing content work, hosting, listing updates, directory ops, routine marketing — classified per this framework

## Why this structure works for Vicki specifically

- **Low entry commitment** — $500/mo is within reach of an independent broker without feeling like a big platform subscription
- **Obvious expansion path** — if the Directory drives measurable lead volume, she adds a second account for the next thing
- **Same model as the existing agreements** — not a bespoke arrangement, it's the same shape as Utopia and the Big Muddy internal retainers, which makes it easier to scale to the next broker (Paul Green, others)
- **Clean classification for expense tracking** — her retainer hours are pre-classified so MBT knows what Directory-supporting work actually costs
