# Asana Onboarding — The Shared Seats

**For:** Whoever opens a shared seat in Asana for the first time.
**Version:** 2026-04-10.
**How long this takes:** Ten minutes.

There are three shared seats in the Hillbilly Dreams Asana workspace. Each one has its own onboarding path. Pick yours and follow it.

---

## Seat 1 — `bigmuddy@chasepierson.tv` (Amy + Tracy)

**Who lives here:** Amy Allen and Tracy Alderson-Allen.
**What this seat is for:** All the business side of Big Muddy — Inn operations, the Deep South Directory product, radio, magazine, touring, shows, bar, housekeeping. If it's about the Natchez operation, it lands here.

### Your first login

1. **Open Asana in your browser** — https://app.asana.com
2. **Sign in with `bigmuddy@chasepierson.tv`** — Chase has the password in Bitwarden and will share it with you verbally.
3. **You will see a list called "My Tasks."** That list is your inbox. Everything assigned to this seat shows up there.
4. **Find the task titled "START HERE — Big Muddy daily."** Pin it if it isn't already pinned. This is the task you check first, every day.

### What you should see on day one

The coordinator has populated this seat with a starting set of tasks. They are all small, all clear, and all designed to walk you through the tools you use every day. Expect something like:

- A welcome task linking to the one-page tour of all Big Muddy properties
- A daily ops checklist for the Inn
- A weekly guest-experience walkthrough (for Amy)
- A weekly revenue check-in (for Tracy)
- A directory listings review (Yelp, Google, Facebook, Airbnb)
- A few "ready for your approval" tasks from Chase

**If any of these are missing or broken, tell Chase.** The coordinator will fix the pattern, not just the task.

### How you work this seat

- **Once a day, open Asana.** Look at the START HERE task. Look at your list. Click the top thing.
- **Every task has a link.** Open the link. Look at the thing. Leave a comment ("approved" or "change X"). Mark the task done.
- **If you create a task for Chase,** make it in this seat and assign it to him. The coordinator will sweep it into his list at 4pm.
- **If you need a human answer right now,** text Chase. Asana is for the things that can wait until 4pm.

### What you never do

- Never assign a task to `amyaldersonallen@gmail.com` or `tracyaldersonallen@gmail.com`. Those are your personal accounts and they are being retired for operational work. Use this shared seat.
- Never mark someone else's task complete. Only your own.
- Never attach a file. Link to where the file lives.
- Never worry about "is this the right project?" The coordinator handles routing. Put it in your own list and let the sweep fix it.

## The START HERE task pattern

Every shared seat has one (and only one) task called "START HERE." That task never gets deleted or replaced — only updated. It's the task you pin to the top of your list and read first every day.

The START HERE task contains:
- The current priorities for this week
- Links to the dashboards, tools, and reference docs you need
- Pointers to any open "waiting for Chase" items
- The most recent housekeeping comment from the coordinator

If you only have time to read one thing in Asana, read the START HERE task.

---

## The daily rhythm

Here is what a normal day should feel like:

- **Morning (you):** Open Asana. Read START HERE. Work down your list. Close what you finish.
- **Throughout the day (Chase):** Chase creates tasks as thoughts come up. They queue up with the coordinator. They do not hit your inbox yet.
- **4:00pm Central (the coordinator):** The daily sweep runs. New tasks land on your list. Stale tasks get archived. A housekeeping comment posts on the START HERE task.
- **Evening (you):** Glance at the housekeeping comment so you know what changed. If anything in tomorrow's list needs a heads-up, reply in the comment.

That is the entire rhythm. It should feel calm. If it ever feels frantic, tell Chase.

## Who to call when it breaks

| Problem | Who to call |
|---|---|
| Can't log in | Chase (he has Bitwarden) |
| Task makes no sense | Leave a comment on the task, then text Chase if urgent |
| Too many tasks / feels frantic | Chase — he will ask the coordinator for a sweep |
| You finished something big and want it off the list | Mark it complete and leave a comment |
| You need a decision from Chase | Create a task on yourself, assign to Chase, set a due date |

**Chase is always the human escalation path.** The coordinator is an AI — it is fast and consistent but it cannot make judgment calls on behalf of the business. Those go to Chase.
