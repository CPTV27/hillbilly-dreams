# Virtual art staging (“Debbie” concept)

Place **art** into **real spaces** without nails first — comps for gallerists, realtors, and venue partners.

## Workflow A — photographic (fast)

1. **Clean plates** of each wall segment (cross-polar if glass).
2. **Lens-matched stills** of artwork (or high-res files from artist).
3. **Composite in Photoshop / Affinity** — perspective warp + shadow paint.
4. Deliver **print + web** masters + layered PSD for revisions.

## Workflow B — 3D (immersive)

1. **Room mesh** from LiDAR / photogrammetry.
2. **Swap materials** on wall planes; UV-align framed pieces.
3. Export **WebXR** + still renders from shared camera positions.

## Deliverables

| Tier | You get |
|------|---------|
| Photo | 3–6 wall comps, 4k JPEG |
| Hybrid | Comps + simple turntable video |
| Full | Comps + `/vr/{slug}` experience with hotspots per piece |

## Pricing (starting bands)

- **$500–1,000** per staging batch (≤6 pieces, one room).
- **+$250** per additional room or lighting scenario.

Empty listing — stage **virtual furniture + art** so the Zillow scroll does not look abandoned.

## Artist use case

Show **how the work lives** in Anthologist, Camp, or a client’s loft before crate ships.
