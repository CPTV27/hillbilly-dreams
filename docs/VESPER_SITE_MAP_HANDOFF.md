## The Design Challenge

Each level needs its own personality but they all need to feel like the same family:

| Level | Aesthetic | Feel |
|---|---|---|
| HDI | Mysterious, playful | "Who built this?" |
| MBT | Light, bright, precise | "This is serious technology" |
| Big Muddy | Dark, photographic, editorial | "This is real culture" |
| DSD | Clean, functional, authoritative | "I trust this to find me a restaurant" |
| Bearsville | Earthy, natural, creative | "The Northeast version" |
| Admin | Utility, Google Lab | "I can get work done here" |

**The thread that connects them:** Gold (#c8943e) as the accent everywhere. "Powered by Measurably Better Things" in every footer. Real photography, never stock. The quality of the imagery is what makes them feel related.

---

## What I Need From You

1. A visual system that defines how each level looks and feels — color, typography, spacing, image treatment
2. Key page layouts for the highest-impact pages (Touring home, Magazine home, DSD listing, MBT platform, Admin dashboard)
3. A component library concept showing how shared elements (headers, footers, cards, buttons) adapt across levels
4. The "Semantic Photo Search" UI for the admin/creator space (V1.2 is a great start)

Take your time. This is the definitive visual identity for the whole ecosystem.
