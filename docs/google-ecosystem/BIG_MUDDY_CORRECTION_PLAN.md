## CRITICAL ISSUES IDENTIFIED

### ISSUE 1: MANAGEMENT FEE POSITION (Financial Architecture)

**Current State (Kimi Output):**
- Studio C Base Management Fee (4%) placed in **Tier 5**
- Sits BELOW Ambassador Compensation Pool (Tier 4)

**Risk:** 
- In slow months, Ambassadors get paid, Studio C gets $0
- Operator cash flow subordinated to staff
- Unsustainable for Studio C operations

**Required Fix:**
```
CURRENT (WRONG):
Tier 1: COGS + OpEx
Tier 2: Capital Partner Preferred Return
Tier 3: Operations Reserve (10%)
Tier 4: Ambassador Compensation (Base + Bonus)
Tier 5: Studio C Base Fee (4%) ← YOU ARE HERE

CORRECTED:
Tier 1: COGS + OpEx + **Studio C Management Fee (4%)**
Tier 2: Capital Partner Preferred Return (8%)
Tier 3: Operations Reserve (10%)
Tier 4: Ambassador **Bonuses Only** (Base moved to Tier 1 OpEx)
Tier 5: Performance Incentives
```

**Impact:** Ensures Studio C receives predictable monthly cash for operations, regardless of occupancy fluctuations.

### ISSUE 3: "FREE ROOM" ACCOUNTING TRAP (Synergy Model)

**Current State:**
- Claims $75K savings by putting artists in Inn rooms "for free"
- No internal transfer pricing

**Risk:**
- Tracy & Amy see this as "Lost Revenue" (opportunity cost)
- Muddies asset valuation
- Creates accounting inconsistency

**Required Fix: Shadow Rate Protocol**

```
INTERNAL TRANSFER PRICING:

Media Co. (Artist Lodging)
├── Pays Inn: $50/night (discounted internal rate)
├── Inn records: Occupancy revenue
├── Media Co. records: Cost of Goods Sold
└── Net effect: Zero cash impact, proper accounting

EXAMPLE:
Artist stays 3 nights
├── Inn P&L: +$150 revenue
├── Media P&L: -$150 COGS
├── Consolidated: $0 (as intended)
└── But: Both entities show accurate economics
```

**Benefits:**
- Inn shows true occupancy and revenue
- Media Co. shows true cost structure
- Asset valuation protected
- No "free lunch" accounting
