## Color Code for Status

When placing images on pages, use these overlays:
- **No overlay** = final, approved, real photo in the right spot
- **BORROWED (teal)** = real photo but from wrong context — usable, should eventually replace
- **AI IMAGE (amber)** = AI-generated, needs real photo replacement
- **PLACEHOLDER (red)** = wrong image entirely, just holding the space

See `apps/web/lib/content-status.ts` for the color system.
