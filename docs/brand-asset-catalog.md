## Delta Dawn Workflow

### How it works:
1. **Tracy sends request** → "I need a poster for [event/artist/promotion]"
2. **Delta Dawn (AI design agent)** receives the request
3. **Delta Dawn searches this catalog** for:
   - Correct brand kit (logo, colors, fonts)
   - Relevant photos from local assets or Google Drive
   - Template styles from existing designs
4. **Delta Dawn generates** poster/graphic using multi-modal AI
5. **Tracy reviews** and approves or requests changes
6. **Final asset** goes to print queue / social media pipeline / website

### Asset priority order:
1. Local processed WebP photos (highest quality, already optimized)
2. Google Drive brand package files (logos, fonts, guidelines)
3. Google Drive photo folders (may need processing)
4. AI-generated supplementary elements (backgrounds, textures — never faces)

### Brand consistency rules:
- Always use official logos from brand package folders
- Match color palette from brand guidelines
- Use designated fonts only
- Real photography takes priority over AI-generated imagery
- Every design must include brand-appropriate contact info
