### MEMPHIS, TN

- **console:** Spectra Sonics 1020-16 (20-input, 8-bus, 2-band EQ); MCI 416B 24-input — the same model used at Muscle Shoals Sound, Criteria (Miami), Willie Mitchell's Royal Studio, and Allen Toussaint's Sea-Saint Studio; MCI 536C mixing console (original; unchanged since Willie Mitchell era); 1969 Spectra Sonics (20×8, 3-band inductor EQ; 100% discrete; handcrafted by Welton Jetton — who also built consoles for Stax Records and Ardent Studios; one of the last Spectra Sonics/Auditronics builds; originally in Canada since 1969); Sphere Eclipse A, 28-input, 16-bus (circa 1970s) — one of ~23–25 known to still exist; Class A transformer-balanced mic preamps; 910/920 graphic EQs; fully restored; API 1604
- **tape:** Studer A827 2" 24-track; MCI JH-110 ½" two-track; Scully 1" 8-track analog; Scully ¼" 2-track; Ampex 351 mono full-track (from estate of Les Paul); Pro Tools with Lynx Aurora converters; MCI JH-24 24-track; Sony 2-track ¼"
- **digital:** Burl B80 Mothership (24-channel AD/DA); Pro Tools HD 2; Pro Tools 192 I/O + 96 I/O; Sony PCM-2500 A/B DAT; Pro Tools 12; Lynx Aurora 16 + 8 (24 analog in/out @ 32-bit/192kHz); Pro Tools; Logic Pro X; Ableton Live 10; UA Apollo 16 + 8
- **iconic_outboard:** 18× Universal Audio 101-D tube preamps (from the original Sam Phillips console); Gates SA39-B tube compressor; 2× Gates Sta-Level tube compressors; UA 176 + 2× UA 175b tube compressors; Teletronix LA-2A; UREI LA-3A; 2× DBX 160VU; 2× Pultec EQH-2; Hycor 4201 passive EQ; **EMT 140 stereo tube plate reverb**; UREI Cooper Time Cube delay
- **echo_chambers:** 3 live echo chambers **designed by Sam Phillips himself**
- **microphones:** 2× Neumann U48; Telefunken U47; Neumann FET 47; Neumann U67; 5× Neumann U87; RCA 77DX ribbon; Altec M11 "Coke Bottle" tube omni; AKG c451e pair; plus EV, Shure, Sennheiser dynamics
- **notable_addition_(2023):** Iconic Stax recording console acquired and installed January 5, 2023 ([samphillipsrecording.com](https://samphillipsrecording.com))
- **key_mics:** Vintage Neumann U48; Neumann U87; 2× RCA DX77 ribbon; Altec 639B (from Les Paul estate); Altec M11 "Coke Bottle" 21D capsule tube mic; AKG 451 vintage matched stereo pair; Neumann U48, U67, 2× U47, 2× U87; 4× RCA 77DX ribbon; 3× AKG C 451E; Sennheiser 441, 4× MD421; 6× Shure SM57; Neumann U67; OG Neumann U47; 2× Royer 121; Royer SF12 stereo; AKG C4000B; Earthworks SR77 pair; Schoeps 221A; 4× Sennheiser 421; 4× Shure SM7b
